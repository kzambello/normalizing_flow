# normalizing_flow
Material for Masked Autoregressive Flow

- normalization_flow.py: Tensorflow implementation of MAF
- MNIST-train.ipynb, MNIST-generate.ipynb: test on the MNIST dataset
- QCD_NF8-train.ipynb, QCD_NF8-generate.ipynb: application to the bulk transition in $N_f=8$ QCD
